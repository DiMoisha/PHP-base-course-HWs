
В КОРЗИНЕ РЕАЛИЗОВАНО УДАЛЕНИЕ 


В папке Site сайт на php. Чтобы не делать гипотетический интернет-магаз, был сделан реально существующий сайт.
Реализовано:
1. Авторизация, аутентификация и регистрация через куки. Время жизни сессии в куках - 1час.
2. Реализовано разделение пользователей на админа и простых смертных. Простой юзер может удалить свой аккаунт, админ - нет.
3. Работа с товарами. Добавление, редактирование, удаление - доступно только админу. 
4. Отзывы. Можно только добавлять.
5. Корзина. Доступна только авторизованным пользователям. Реализованf через БД.

Бэкап БД в файле abz.sql
